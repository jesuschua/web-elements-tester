# Web Elements Tester

A simple black and white web elements tester built with HTML, CSS, and JavaScript.

## Features

- Static text
- Static image
- Dynamic text (current date and time)
- Radio buttons
- Dropdown selector
- Text transformation (hex, binary, random string append)

## Deployment to GitHub Pages

1. Push this repository to GitHub
2. Go to Settings > Pages in your GitHub repository
3. Select the branch (usually `main` or `master`)
4. Select the folder (usually `/ (root)`)
5. Click Save
6. Your site will be available at `https://[username].github.io/web-elements-tester/`

## Local Development

Simply open `index.html` in a web browser or use a local server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

Then open `http://localhost:8000` in your browser.
